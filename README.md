# nitro2

## Build using Docker
` docker build -t xt/nitro2 .`

` docker run -p 3000:3000 xt/nitro2` 

## steps to Build Locally & run

`npm i`

`npm run install:apps`

`npm run build:apps`

`npm run start`

on the browser `http://localhost:3000/home`
